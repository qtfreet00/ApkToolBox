# ApkToolBox Android一体式工具箱

##### 主要功能
* 反编译apk，回编译apk，支持批量
* 反编译，回编译dex
* 一键获取adb shell am start命令
* 集成jadx，jd-gui
* 支持自定义签名
* 支持app查壳
* 支持odex反编译

##### 签名工具来自网易，签名速度比一般签名快50%~80%

###### 更新日志
   移步Release

PS:注意在使用时，尽量不要使用中文路径，不然可能导致编译或者签名失败
